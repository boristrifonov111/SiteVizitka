# Сайт-визитка "Путешествие с любовью"

Одностраничный сайт-визитка для экскурсий по Ленинградской области.

## Структура проекта

- `index.html` - основная HTML страница
- `styles.css` - стили сайта
- `script.js` - генерация QR-кода
- `vercel.json` - конфигурация для Vercel

## Деплой на Vercel

### Вариант 1: Через веб-интерфейс Vercel

1. Зайдите на [vercel.com](https://vercel.com) и войдите в аккаунт (можно через GitHub)
2. Нажмите "Add New Project"
3. Импортируйте репозиторий или загрузите файлы через drag & drop
4. Vercel автоматически определит статический сайт и задеплоит его
5. После деплоя вы получите URL вида `your-project.vercel.app`

### Вариант 2: Через Vercel CLI

1. Установите Vercel CLI:
```bash
npm i -g vercel
```

2. Войдите в аккаунт:
```bash
vercel login
```

3. Задеплойте проект:
```bash
vercel
```

4. Для продакшн деплоя:
```bash
vercel --prod
```

## QR-код

QR-код генерируется автоматически при загрузке страницы и содержит URL текущего сайта. После деплоя на Vercel QR-код будет содержать актуальный URL сайта.
